# mlita_3 — ИДЗ-3 (синтаксический анализатор)

Запуск:
```bash
python3 main.py
```

Язык:
- правильные скобочные записи с двумя видами скобок: `()` и `[]`
- доп. ограничение: не более двух одинаковых скобок подряд (например, `(((` и `]]]` запрещены)

Парсер:
- рекурсивный спуск по LL(1)-грамматике для базовой ПСП
- ограничение по подряд идущим одинаковым терминалам проверяется при чтении символов
